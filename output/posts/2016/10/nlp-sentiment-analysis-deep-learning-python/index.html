Deep Learning for NLP - Sentiment Analysis using Python, Keras and Word2Vec

// tags deep learning machine learning python nlp keras

Sentiment Analysis refers to the use of natural language processing techniques to identify and extract subjective information in text data. A typical sentiment analysis scenario is to identify whether a piece of writing is positive, negative or neutral. Sentiment analysis has many applications ranging from marketing and customer service, to politics and sociology.

The goal of this blog post is to give you a hands-on introduction to sentiment analysis. To do this, we will build a deep learning algorithm for classifying comments about movies into positive or negative sentiments.

This post is divided into 3 parts. We will start by downloading the data and defining the problem. The second part covers some core concepts about deep learning applied to natural language processing. We will look into one deep learning algorithm called recurrent neural network (RNN) and two of its derivatives, namely LSTM and GRU. We will also cover a technique for extracting features from text data called word embedding and one of its most popular algorithms called Word2Vec. The last part is structured in a hands-on tutorial format. We will build a few models using different RNN architectures and text features, and will assess their performances. Our best model will be able to achieve a classification accuracy of 88%.

By the end of this tutorial, you will understand how to build a sentiment analysis algorithm and how to work with text data. You will also get familiar with recurrent neural networks, and the steps to implement them using Python and Keras.

1. Problem Definition

In this tutorial, we will be using a dataset from Kaggle. The dataset is comprised of 25,000 images of dogs and cats.
